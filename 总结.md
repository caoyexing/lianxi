# dom对象的意义是用js操作页面的样式


#### axios
1. Access-Control-Allow-Origin: * 后端cros设置 所有的网址都可以进行访问 没有跨域的限制
```js
  axios.get('url').then(()=>{})
```
2. Access-Control-Allow-Origin: * 也是 *  但是要带有请求头的设置 
```js

  axios({
    url:'',
    headers:'' // 请求头中的参数
    // 默认是 get
    // methods:get
  }).then((res)=>{})

```
3. 反向代理的配置 axios   
```js
  axios.get('url').then(()=>{})

  module.export={
    devServer:{
      proxy:{
        // 直接是.com 后面的网址 不用替换了
        '/ajax':{
          target:'http//www.baidu.com',
          changeOrigin:true,
        }
      }
    }
  }

```
